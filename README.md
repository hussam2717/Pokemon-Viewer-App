# Pokemon-Viewer-App
A simple and interactive Pokémon Viewer built using HTML, CSS, and JavaScript.   Search any Pokémon by name or ID and get instant details with a clean UI.

Live Demo - 

## 🚀 Features
- 🔍 Search Pokémon by name or ID
- 🎲 Random Pokémon generator
- 🖼️ Displays Pokémon image
- 📊 Shows Pokémon ID and type
- ⚡ Fast and responsive UI
- 🎯 Predefined quick search buttons

 ## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- PokéAPI (for fetching data)
